# nginx-http3
Nginx with http3

# How to Install
<h3>Intel Architecture</h3>
% git clone https://github.com/raihan001/nginx-http3.git <br>
% cd nginx-http3 <br>
% sudo chmod +x http3-nginx.sh <br>
% sudo ./http3-nginx.sh<br>
<hr>
<h3>Arm64 Architecture</h3>
% git clone https://github.com/raihan001/nginx-http3.git <br>
% git checkout ARM64 <br>
% cd nginx-http3 <br>
% sudo chmod +x http3-nginx.sh <br>
% sudo ./http3-nginx.sh
